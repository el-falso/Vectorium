sub

function

sub(_vector)

Description

Substracts the specified vector from this vector.

Arguments

Name Type Description
_vector vec3 The vector being substracted.

Returns

vec3 self

Example

vec1 = new Vec3(10, 10, 10);
vec2 = new Vec3(8, 6, 9);
vec1.sub(vec2);
show_debug_message(vec1); // Result: { x : 2, y : 4, z : 1 }
